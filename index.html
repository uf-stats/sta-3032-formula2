One Sample Inference - Nonparametric Tests

Sign test statistic:

B = number of observations strictly greater than the hypothesized null p percentile value

B ∼ H 0 Bin ( n , 1 − p )

Wilcoxon signed-rank test:

  1. Calculate the differences d i = x i − μ 0 for each observation, i.e. center data according to H 0 .
  2. Discard any d i = 0 (as long as they are not more that 10% of the data, otherwise research "adjusted" methods).
  3. Rank | d i | (i.e. rank ignoring the sign).
  4. Calculate the test statistic S + = sum of the ranks corresponding to positive d i 's.

Two Sample Inference - Population Mean

Two sided C.I. (means):

x _ − y _ ∓ t 1 − α / 2 , ν s X 2 n X + s Y 2 n Y

where

ν = ( s X 2 n X + s Y 2 n Y ) 2 ( s X 2 / n X ) 2 n X − 1 + ( s Y 2 / n Y ) 2 n Y − 1

Test statistic:

T.S. = x _ − y _ − Δ 0 s X 2 / n X + s Y 2 / n Y ∼ H 0 t ν

If σ X 2 = σ Y 2 , then replace s X 2 and s Y 2 with

s p 2 = ( ( n X − 1 ) n X + n Y − 2 ) s X 2 + ( ( n Y − 1 ) n X + n Y − 2 ) s Y 2

and ν = n X + n Y − 2 .

Two Sample Inference - Population Proportion

Two sided C.I. (proportions):

p ~ X − p ~ Y ∓ z 1 − α / 2 p ~ X ( 1 − p ~ X ) n ~ X + p ~ Y ( 1 − p ~ Y ) n ~ Y

where n ~ X = n X + 2 , n ~ Y = n Y + 2 , and p ~ X = ( x + 1 ) / n ~ X , p ~ Y = ( y + 1 ) / n ~ Y .

Test statistic:

T.S. = p ~ X − p ~ Y − Δ 0 p ~ X ( 1 − p ~ X ) n ~ X + p ~ Y ( 1 − p ~ Y ) n ~ Y ∼ H 0 N ( 0 , 1 )

Two Sample Inference - Population Variance

( s X 2 s Y 2 1 F ( 1 − α / 2 ; n X − 1 , n Y − 1 ) , s X 2 s Y 2 1 F ( α / 2 ; n X − 1 , n Y − 1 ) )

T.S. = s X 2 s Y 2 Δ 0 ∼ H 0 F n X − 1 , n Y − 1
